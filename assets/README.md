# Assets Directory

This directory contains visual assets for the JCUSSDLib documentation.

## Images

1. **architecture-diagram.png** - The VoIP USSD approach architecture diagram
2. **android-banner.png** - The Android USSD automation banner

## Instructions

To add the images:
1. Save both screenshots as:
   - `architecture-diagram.png`
   - `android-banner.png`
2. Place them in this `assets/` directory
3. The README.md will automatically display them
